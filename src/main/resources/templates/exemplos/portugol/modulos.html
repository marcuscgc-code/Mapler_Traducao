Módulos

Módulos são blocos externos ao bloco principal, podem ser executados várias vezes apenas com uma chamada. Assim, módulos evitam que um mesmo algoritmo precise ser escrito várias vezes.

Sintaxe

                            variaveis

                                // Declaração do módulo

                                dizerOla: modulo;


                            inicio

                                // Chamada do módulo

                                dizerOla;


                            fim



                            // Definição do módulo

                            modulo dizerOla; 

                                escrever "Olá, mundo!";

                            fim modulo;

Observação!

  • O módulo precisa ser declarado no bloco principal.
  • Um módulo reconhece as variáveis que pertecem ao bloco principal.
  • Um módulo pode ter diversos comandos internos.

Exemplo

                            variaveis

                                a, b:inteiro;

                                somar: modulo;

                                dividir: modulo;

                                sibtrair: modulo;

                                multiplicar: modulo;


                            inicio

                                a<-10;

                                b<-5;



                                somar;

                                sibtrair;

                                multiplicar;

                                dividir;


                            fim



                            modulo somar 

                                escrever a, "+", b, "=", a+b;

                            fim modulo;



                            modulo multiplicar 

                                escrever a, "*", b, "=", a*b;

                            fim modulo;



                            modulo sibtrair 

                                escrever a, "-", b, "=", a-b;

                            fim modulo;



                            modulo dividir 

                                escrever a, "/", b, "=", a/b;

                            fim modulo;
