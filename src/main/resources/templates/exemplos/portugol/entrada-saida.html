Comandos de Entrada e Saída

Comando usados para interação do usuário com o algoritmo.

Entrada

Recebe dados do usuário a partir do teclado. O valor digitado será armazenado na variável indicada.
ler <variavel>; 

                        // Exemplo do comando de entrada

                        variaveis

                            numero: inteiro;

                        inicio

                            ler numero;

                        fim

Saída

Exibe o conteúdo no terminal. O comando aceita um texto ou uma variavel como parâmetro. Pode-se também concatenar um texto com uma variável.
escrever <"texto"><, variavel>; 

                        // Exemplo do comando de saída

                        variaveis

                        inicio

                            escrever "Hello, world!";

                        fim

Exemplo

                        // Exemplo utilizando comandos de entrada e saída

                        variaveis

                            nome: cadeia;

                        inicio

                            escrever "Digite seu nome:";

                            ler nome;

                            escrever "Olá ", nome;

                        fim
