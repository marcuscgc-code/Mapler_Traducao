O ponto de partida do fluxograma é simbolizado pela figura de início, que representa o ponto inicial do algoritmo. Essa figura é sempre utilizada como o ponto de partida lógico para o desenvolvimento do processo.

A figura de encerramento, representada como 'Fim', sinaliza o término do programa. Quando a lógica alcança este ponto, o código é finalizado.

O símbolo 'Entrada' indica que o usuário deve fornecer um valor, o qual o algoritmo irá ler e armazenar em uma variável. Essa figura é utilizada para solicitar que o usuário insira informações essenciais para o prosseguimento do algoritmo.

O símbolo 'Saída' representa a forma como o algoritmo se comunica com o usuário do sistema, exibindo informações no terminal ou visor. Geralmente, é usado para que o sistema compartilhe resultados ou mensagens importantes com o usuário, facilitando a interação entre ambos.

O símbolo 'Processamento' engloba as figuras relacionadas a cálculos e alterações de valores no fluxograma. É usado para representar ações como atribuições, operações matemáticas e outras interações com variáveis, indicando o processamento de dados e informações dentro do algoritmo.

A figura 'Decisão' é a única no fluxograma que pode ter dois caminhos saindo dela. Um valor, que pode ser configurado automaticamente ou pelo desenvolvedor, é definido para determinar qual caminho será percorrido com base no resultado da condição presente na decisão.
