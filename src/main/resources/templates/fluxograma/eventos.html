No software Mapler, a criação e edição de fluxogramas são facilitadas por um conjunto de três botões essenciais que desempenham funções específicas:

Função

O botão 'Mover' é projetado para permitir o deslocamento de elementos individuais do fluxograma para diferentes posições dentro do diagrama.

Utilização

Isso é particularmente útil quando se deseja ajustar a disposição dos elementos, organizar o fluxo lógico de maneira mais clara ou reposicionar caixas de decisão, entrada, saída ou outras figuras conforme necessário.

Função

O botão 'Associar' tem como finalidade estabelecer conexões entre os elementos do fluxograma, criando linhas ou setas que representam a sequência lógica das operações.

Utilização

Ao clicar no botão 'Associar' e selecionar duas figuras no fluxograma, você pode criar uma linha que indica a relação entre esses elementos, ajudando a definir a ordem de execução das ações no algoritmo.

Função

O botão 'Remover' permite a exclusão de elementos indesejados do fluxograma, como figuras ou conexões entre elas.

Utilização

Ao clicar no botão 'Associar' e selecionar duas figuras no fluxograma, você pode criar uma linha que indica a relação entre esses elementos, ajudando a definir a ordem de execução das ações no algoritmo.
